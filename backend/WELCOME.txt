⚡ DÉMARRAGE RAPIDE
═══════════════════════════════════════════════════════════════════════════

  Option 1 : Tout Automatique (Recommandé)
  ─────────────────────────────────────────
  npm run setup:full

  Option 2 : Étape par Étape
  ──────────────────────────
  npm run setup              # Installer + générer Prisma
  npm run db:setup           # Configurer PostgreSQL
  npm run prisma:migrate     # Créer le schéma
  npm run db:seed            # Données de test (optionnel)
  npm run dev                # Démarrer

  Option 3 : Avec Vérifications
  ─────────────────────────────
  npm run start:check        # Vérifie tout et démarre

═══════════════════════════════════════════════════════════════════════════

📋 COMMANDES ESSENTIELLES
═══════════════════════════════════════════════════════════════════════════

  Développement
  ─────────────
  npm run dev                # Démarrer en mode développement
  npm start                  # Démarrer en mode production
  npm run start:check        # Démarrer avec vérifications

  Base de Données
  ───────────────
  npm run db:setup           # Configurer PostgreSQL
  npm run prisma:generate    # Générer Prisma Client
  npm run prisma:migrate     # Créer/appliquer migrations
  npm run db:seed            # Peupler avec données de test
  npm run prisma:studio      # Interface graphique DB

  Utilitaires
  ───────────
  npm test                   # Lancer les tests
  npm run lint               # Vérifier le code
  npm run db:reset           # Réinitialiser la base
